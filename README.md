# SPLT - Workout Split Organizer

A React-based fitness application that helps you organize your workout splits and provides customized exercises per muscle group.

## Features

- **Workout Split Organization**: Create and manage different workout split routines
- **Muscle Group Focus**: Organize exercises by target muscle groups
- **Customizable Routines**: Tailor workouts to your fitness goals
- **Easy Navigation**: Simple and intuitive user interface

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

## Available Scripts

### `npm start`

Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.

### `npm test`

Launches the test runner in interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder.\
The build is optimized for the best performance.
